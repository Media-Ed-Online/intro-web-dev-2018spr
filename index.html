Welcome to Introduction to Web Design & Development!

In this course, students will gain valuable skills in this introduction to the fundamentals of website development, content design, and navigation, with areas of focus in directory structure, graphic design, navigation, and audio/video integration.

Get Started

Is this the class for me?

Absolutely! While challenging, this course is open to all university students and geared to non-majors. We start at the beginning, and spend much of our time with the web's basic languages; HTML and CSS.

We use several resources in this class. Make sure you check out our Moodle Page which contains the syllabus and schedule for this semester, and where you’ll receive your grades. You will also need to continually view this site’s GitHub Repo as you’ll access our Issues Board, Projects, and Wiki for each assignment.

Jekyll also happens to be the engine behind GitHub Pages, which means you can use Jekyll to host your project’s page, blog, or website from GitHub’s servers for free.
